# Changelog

## [1.0.0] - 2026-02-16

### Added
- **Core Printing Support**: Full integration with Phomemo T02 thermal printers via Bluetooth LE.
- **Image Processing Engine**: Advanced dithering algorithms to optimize images for 1-bit printing:
  - Floyd-Steinberg (Error Diffusion)
  - Halftone (Dot Screen)
  - Threshold (Simple Binarization)
- **Camera Integration**: Capture photos directly within the app for instant printing.
- **Photo Library Access**: Select and print existing photos from the device gallery.
- **Theme Customization**: Added "Pink Mode" toggle in settings (you can toggle this if you have the pink printer variant).
- **Bluetooth Sharing**: Experimental feature to share printer access between devices (Host/Client mode).
- **Smart Orientation**: Automatic image rotation and resizing to fit the 58mm paper width.
